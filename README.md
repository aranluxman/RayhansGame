# Treasure Scramble

A single-file browser board game for 2 to 4 players.

Open `index.html` in a browser to play locally, or deploy the repository as a static site on GitHub Pages, Cloudflare Pages, Netlify, or another static host.

## Gameplay

- Roll one die and move around the board.
- Collect 5 treasures.
- Reach or pass Start after collecting 5 treasures to enter the center Treasure Temple and win.
- Spaces include treasures, traps, mini-games, chaos cards, trades, and teleports.
